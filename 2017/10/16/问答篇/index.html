问答篇

发表于 2017-10-16 | 分类于 问答

此篇为日常问答篇，收集一些问题（啥都有）。目的是一句话说明白。

1.invalidate() 和 postInvalidate() 有什么区别？
答：post的区别，还不明显吗？

2.Looper 中的 MessageQueue 是什么队列？
答：既然是队列，你说呢？数据元是 Message 的链表。

3.Looper 可以关联多个 Handler 吗？
答：其实 Handler 关联的是 Message，Looper 里面是 MessageQueue，Looper -> MessageQueue ->多个Message ->每个 Message 关联一个 Handler 的间接关联。所以可以多关联.

4.Handler A 发送的消息可以分发给 Handler B 吗？
答：其实理论上应该是可以的。但是实际上不行，在 sendMessage 后，在 Handler 基类中为 Message 设置了一个target 值为自身，无论是否在之前开发者设置值。

5.家里的 VPN 突然连不上了又多少可能？
1.VPN 设备坏了。
2.公网 Ip 失效了。
3.电信封了端口。
