正则（匹配）表达式

发表于 2017-10-18 | 分类于 Java

与其叫正则表达式，不如叫做匹配表达式。

1.在正则表达式中，括号为分组效果。| 为或操作。

2.可以使用 boolean String.matches(String regex) 判断字符串是否匹配。也可以使用 java.util.regex.Pattern 类的 static boolean Patter.matches(String regex,CharSequece input).

  1. “\G” 代表前一个匹配结束.

4.贪婪型：贪婪表达式会为所有可能的表达式发现尽可能多的匹配。
勉强型：匹配满足模式所需的最少字符数。后面带个 “?”
占有型：防止回溯

5.可以使用 Pattern Pattern.compiler(String regex) 来编译你的正则表达式。然后使用 Pattern.matcher(String input) 方法生成一个 Matcher 对象。

6.Group:用括号表示分组.如 “A(B(C))D”， 组0是 “A(B(C))D” ；组1是 “B(C)”；组2是 “C”
然后匹配的一个结果中使用 group(int grop) 可以返回相应分组匹配的结果。

7.Pattern 标记.用以调整匹配行为. Pattern.compile(String regex,int flag) 看文档吧。

8.Matcher.reset(String input) 替换目的字符串.
